**E-Medical Record System (EMR)**
A simple web-based system that turns paper medical records into digital records.

**What it does:**
Lets patients book appointments online
Doctors can write digital prescriptions
Lab reports can be uploaded and viewed online
Different dashboards for admins, doctors, and patients
Keeps all medical records safe and organized in one place

**Who can use it:**
**Patients**: Book appointments, see prescriptions, view lab reports
**Doctors**: See patient records, write prescriptions, check appointments
**Admins**: Manage all users, control the system
**Lab Staff**: Upload test results

**What I used to build it:**
HTML, CSS, JavaScript for the website
Visual Paradigm and Draw.io for making diagrams
Microsoft Office for documentation

**How it works:**
Users login (patients, doctors, or admins)
Each user sees their own dashboard
Patients can book appointments
Doctors can write prescriptions
Lab staff can upload reports
Everything is saved securely in the system

**Main features:**
Secure login system
Easy appointment booking
Digital prescriptions
Online lab reports

Clean, easy-to-use design

This project helps hospitals move from paper files to digital records, making everything faster, safer, and more organized.
